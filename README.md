# Don Atento: Realidad Cognitiva Inmobiliaria 🚀

**Don Atento** es una plataforma de vanguardia diseñada para la gestión inteligente de activos inmobiliarios, combinando Inteligencia Artificial generativa, visualización 3D avanzada (Gaussian Splatting) y analítica predictiva.

## 🌟 Características Principales

- **Digital Twins (Fase 3)**: Reconstrucción 3D de inmuebles mediante Gaussian Splatting con interacción "Splat-Click" para inspecciones espaciales.
- **Orquestador IA (Fase 2)**: Negociación autónoma vía WhatsApp para reparaciones y atención a inquilinos.
- **Mantenimiento Predictivo (Fase 4)**: Algoritmos de "Health Score" para predecir fallas críticas en infraestructura.
- **Analítica de ROI (Fase 2)**: Simuladores financieros en tiempo real con estética Cyber-Tech.
- **Multitenancy & Super Admin (Fase 5)**: Arquitectura escalable para gestionar múltiples inmobiliarias desde un panel central de control.

## 🛠️ Stack Tecnológico

- **Frontend**: Next.js 14, React Three Fiber (3D), Tailwind CSS, Recharts, Lucide Icons.
- **Backend**: Node.js, Prisma ORM, PostgreSQL.
- **IA**: LLMs personalizados, Computer Vision para detección de anomalías en video.

## 🚀 Inicio Rápido

1. **Instalación**:
   ```bash
   cd frontend && npm install
   cd ../backend && npm install
   ```
2. **Base de Datos**: Configura tu `.env` en el backend y ejecuta:
   ```bash
   npx prisma migrate dev
   ```
3. **Desarrollo**:
   ```bash
   npm run dev
   ```
